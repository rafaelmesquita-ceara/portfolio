Sobre

Olá! Me chamo Rafael, sou acadêmico de engenharia da computação pela Universidade Federal do Ceará, Sobral (UFC). Atuando como freelancer desde 2018, desenvolvendo aplicações web e desktop para empresas de pequeno e médio porte.


Tive inicio na programação desde 2016, onde, por meio do ensino técnico em informática, descobri como era a forma que eram desenvolvidas as aplicações que vemos atualmente, desde então, sigo com bastante entusiasmo e amor pela área, buscando sempre aperfeiçoar meus conhecimentos, conhecer boas práticas e superar minhas habilidades a cada dia


Sempre encarando novos desafios, com muita motivação e de maneira inovadora, porque sei que mesmo não dando certo, serei recompensado do mesmo jeito em forma de experiência

Projetos

  • Note_APP

    O note_app é um simples gerenciador de notas feito em C# utilizando o entity framework, o projeto foi feito para fins de aprendizado. Conceitos como conexão e permutação do banco de dados foram utilizados de forma prática nessa solução.

  • BookListMVC

    O BookListMVC é um simples gerenciador de livros feito em ASP.NET CORE 2.1 no modelo MVC utilizando o Entity Framework como ORM, o projeto foi feito para fins de aprendizado.

  • AspNetCoreAPI

    O AspNetCoreAPI é uma API simples feita em ASP.NET CORE 3.1 utilizando o Entity Framework como ORM, o projeto foi feito para fins de aprendizado.

  • Sys Franthesk

    Sistema gerencial de uma empresa real feito em C# utilizando Entity Framework como ORM.

  • Simple_Ecommerce

    Sistema simples para um ecommerce(no momento está apenas com o backend) utilizando Node.JS e Knex.JS como ORM. Conceitos como JWToken, Integração com os correios, Migrations e o modelo API do Node.JS foram aplicados.

  • Simple_Upload

    Sistema simples para upload de arquivos, backend feito em NodeJS e frontend feito em ReactJS, conceitos como Multer.JS, Componentes React.JS e requisições para APIs em React.JS foram aplicados.
